Simulation Add In

This tool is to support using Excel as a front end for AutoMod(TM) simulation models.

Generally, it is expected that the model already exists when starting this tool, but that is not required.

The first thing you will want to do (after adding the add-in to Excel) is to initialize a new project from the ribbon command Initialize.

Quick Start

You can set up a new UI in just a few quick steps:

1) open a new Excel file

2) Click on "Initialize"

3) Identify the path that contains the models .exe file

4) Select the model itself

5) Save the Excel file

6) Select Run
